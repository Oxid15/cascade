cascade.models¶

class cascade.models.Model(**kwargs)¶

Base class for any model. Used to provide unified interface to any model, store metadata including metrics.

evaluate(*args, **kwargs) → None¶

Evaluates model against any metrics. Should not return any values, just populating self.metrics dict.

fit(*args, **kwargs)¶

Method to encapsulate training loops. May be provided with any training-related arguments.

load(filepath) → None¶

Loads model from provided filepath. May be unpickling process or reading json configs. Does not return any model, just restores internal state.

predict(*args, **kwargs)¶

Method to encapsulate inference. May include preprocessing steps to make model self-sufficient.

save(filepath) → None¶

Saves model’s state using provided filepath.




class cascade.models.ModelLine(folder, model_cls=<class 'cascade.models.model.Model'>, meta_prefix=None)¶

A manager for a line of models. Used by ModelRepo for access to models on disk. A line of models is typically a models with the same hyperparameters and architecture, but different epochs or using different data.

save(model: cascade.models.model.Model) → None¶

Saves a model and its meta data to a line folder. Model is automatically assigned a number and a model is saved using Model’s method save. Name is assigned using f’{idx:0>5d}’. For example: 00001 or 00042. The name passed to model’s save is without extension. It is Model’s responsibility to correctly assign extension and save its state.




class cascade.models.ModelRepo(folder, model_cls=<class 'cascade.models.model.Model'>, meta_prefix=None)¶

An interface to manage experiments with several lines of models. When created, initializes an empty folder constituting a repository of model lines.
